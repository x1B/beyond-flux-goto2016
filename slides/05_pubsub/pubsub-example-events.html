PubSub (LaxarJS) - Example - Events

- the components are *loosely coupled* allowing for wiring at configuration time - in order to connect them, their source code does not need to be touched - declarative JSON wiring allows to statically reason about the connections - connections are formed by specifying a common topic for features of view- and activity components - actions may be triggered by multiple sources, resources always have exactly one *owner*-activity. - There are tools that use this information to produce a graphical overview of your application.
  • JSON-Seitenkonfiguration
  • Deklarative Zuordnung von Kollaborateuren
  • Statisch, aber beliebig erweiterbar (z.B. 2 shopping-carts)
